# Customer Management Module

Customer relationship management for ERPlora Hub.

## Features

- Customer database with contact information
- Purchase history tracking
- Customer statistics and analytics
- Search and filtering
- Import/export capabilities

## Installation

This module is installed automatically via the ERPlora Marketplace.

## Configuration

No additional configuration required. The module works out of the box.

## Usage

Access via: **Menu > Clientes**

### Customer Management

- **Add Customer**: Create new customer records
- **View History**: See purchase history per customer
- **Statistics**: View customer spending patterns

## Models

| Model | Description |
|-------|-------------|
| `Customer` | Customer profile with contact info |

## Permissions

| Permission | Description |
|------------|-------------|
| `customers.view_customer` | View customer list |
| `customers.add_customer` | Create customers |
| `customers.change_customer` | Edit customers |
| `customers.delete_customer` | Delete customers |
